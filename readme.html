# 🧙‍♂️ Hogwarts: A Magia das Quatro Casas Bem-vindo ao repositório do projeto **Hogwarts: A Magia das Quatro Casas**, um site desenvolvido em **WordPress** com foco na apresentação interativa das casas da escola de magia Hogwarts, do universo Harry Potter. O sistema foi projetado para oferecer uma experiência imersiva, organizada e visualmente atrativa para fãs da franquia, explorando história, características e curiosidades de cada casa. --- ## 🚀 Funcionalidades Principais * **Página Inicial (Home)**: Apresentação geral das casas de Hogwarts * **Páginas Individuais**: - 🦁 Grifinória - 🐍 Sonserina - 🦅 Corvinal - 🦡 Lufa-Lufa * **Conteúdo Completo por Casa**: - História - Fundador - Características - Personagens notáveis - Curiosidades * **Design Responsivo**: Adaptado para diferentes dispositivos * **Interface Intuitiva**: Navegação simples e organizada * **Identidade Visual Temática**: Inspirada no universo Harry Potter --- ## 🛠️ Tecnologias Utilizadas * **CMS**: WordPress * **Tema**: Astra Child * **Plugins**: - Editor Clássico - Spectra --- ## 🎨 Design e Identidade Visual ### Tipografia * **Body**: Cormorant Garamond * **Headings**: Playfair Display ### Paleta de Cores * Dourado: `#ffc500`, `#efb810` * Vermelho: `#7f0909` * Verde: `#4e653d`, `#20331b` * Azul: `#194569`, `#5f84a2` * Amarelo: `#f9db5c` * Preto: `#000000` --- ## 📂 Estrutura do Projeto ``` projeto/ ├── wp-admin/ ├── wp-content/ │ ├── themes/ │ ├── plugins/ │ └── uploads/ ├── wp-includes/ ├── wp-config.php └── index.php ``` --- ## ⚙️ Pré-requisitos * XAMPP (ou outro servidor local) * PHP 7.2 ou superior * MySQL ou MariaDB * Navegador web --- ## 🚀 Configuração e Execução 1. **Clonar ou copiar o projeto para o diretório do XAMPP**: ```bash C:\xampp\htdocs\hogwarts-quatro-casas ``` 2. **Iniciar o XAMPP**: - Apache ✅ - MySQL ✅ 3. **Acessar o site**: ``` http://localhost/hogwarts-quatro-casas ``` 4. **Acessar o painel administrativo**: ``` http://localhost/hogwarts-quatro-casas/wp-admin ``` --- ## 🔐 Acesso ao Sistema (Ambiente Local) * **Usuário**: erik0301 * **Senha**: Erik0301?
