🧙‍♂️ Hogwarts: A Magia das Quatro Casas

Um site temático desenvolvido em WordPress inspirado no universo de Harry Potter.

📖 Sobre o Projeto

O projeto Hogwarts: A Magia das Quatro Casas foi desenvolvido com o objetivo de apresentar as quatro casas de Hogwarts de forma interativa e organizada:

  • 🦁 Grifinória
  • 🐍 Sonserina
  • 🦅 Corvinal
  • 🦡 Lufa-Lufa

Cada casa possui uma página dedicada com informações sobre sua história, valores, personagens e curiosidades.

🎯 Objetivo

  • Apresentar o universo de Harry Potter
  • Criar uma navegação simples e intuitiva
  • Proporcionar uma experiência visual imersiva
  • Servir como projeto prático de WordPress

🛠️ Tecnologias Utilizadas

  • WordPress
  • Tema: Astra Child
  • Plugins:
    • Editor Clássico
    • Spectra

🧩 Funcionalidades

  • Estrutura com múltiplas páginas
  • Design responsivo
  • Conteúdo organizado por casas
  • Interface intuitiva

🚀 Como Executar o Projeto

  1. Instale o XAMPP
  2. Ative Apache e MySQL
  3. Coloque o projeto em: 
    C:\xampp\htdocs\hogwarts-quatro-casas
  4. Acesse: 
    http://localhost/hogwarts-quatro-casas
  5. Painel admin: 
    http://localhost/hogwarts-quatro-casas/wp-admin
